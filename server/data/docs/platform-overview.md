# Eventforce Platform Overview

## Assistant coverage

Eventforce AI Assistant helps event teams answer questions about registration, badge pickup, PhotoApp, WhatsApp messaging, feedback management, CRM exports, sponsor ROI, agenda reminders, parking, and onsite support.

## RAG and web search

The assistant retrieves answers from approved event documents first. If the local knowledge base does not contain enough information and web search is enabled, the assistant can use Tavily for live web search. If neither path returns a confident answer, the assistant should create a ticket and share the ticket number with the user.
