# WhatsApp Messaging Playbook

## Event reminders

WhatsApp reminders can be scheduled for registration confirmation, badge pickup, agenda changes, session starts, speaker updates, and post-event feedback. Messages can include dynamic attendee fields such as name, ticket type, room, and selected track.

## Two-way support

Attendees can reply to WhatsApp messages for quick support. The assistant should answer questions from approved event documents and escalate unresolved or account-specific issues to the event operations team.

## Broadcast safety

All broadcast messages require opt-in status, approved templates, and rate-limit checks. Sensitive attendee data should not be sent in open message threads.
