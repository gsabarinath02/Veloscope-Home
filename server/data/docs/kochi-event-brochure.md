# Kochi Half Marathon 2026 Event Brochure

## Event overview

Kochi Half Marathon 2026 will be held on Sunday, 20 September 2026 at Marine Drive Ground, Kochi, Kerala. The event is organized by Veloscope Events Pvt Ltd and powered by the Eventforce event technology platform. The event theme is Run the Coast and the route passes Marine Drive, High Court Junction, MG Road, and Subhash Bose Park before returning to the finish area.

## Race categories and fees

The available race categories are Half Marathon 21.1K, Open 10K, Fun Run 5K, and Family Run 3K. The Half Marathon registration fee is INR 1,499, the Open 10K fee is INR 999, the Fun Run 5K fee is INR 599, and the Family Run 3K fee is INR 399. Fees include bib, timing chip where applicable, event T-shirt, finisher medal, breakfast, hydration, and digital certificate.

## Registration status

Registration is currently open for all race categories. Online registration opened on 1 June 2026 and closes on 5 September 2026 or earlier if slots sell out. Spot registration is not planned for race weekend.

## Start times and cut-off times

The Half Marathon starts at 5:30 AM and has a cut-off time of 3 hours 30 minutes. The Open 10K starts at 6:15 AM and has a cut-off time of 1 hour 45 minutes. The Fun Run 5K starts at 7:00 AM and has a cut-off time of 1 hour. The Family Run 3K starts at 7:20 AM and has a cut-off time of 45 minutes.
