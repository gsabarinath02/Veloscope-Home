# PhotoApp User Guide

## Downloading event photos

Attendees can download event photos from the PhotoApp. Open the app and enter the registered email address to access the gallery. Photos are grouped by day, room, and branded sponsor zone. High-resolution downloads are available for approved attendees and sponsors.

## Branded galleries

PhotoApp supports sponsor-branded galleries, watermarking, moderated uploads, and private albums. The event team can publish gallery links to email, WhatsApp, and the event app within seconds.

## Media privacy

Restricted media collections require approved access. If an attendee cannot access a private gallery, the support team should verify the registered email and permission group before sharing a download link.
