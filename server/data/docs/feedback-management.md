# Feedback Management

## Feedback capture

Feedback can be collected through the event app, QR codes, WhatsApp links, and post-session surveys. The recommended post-session survey asks for a rating, a topic relevance score, and one optional comment.

## Sentiment and action tracking

The feedback dashboard groups comments by sentiment, topic, room, speaker, and attendee segment. Low ratings can automatically create action items for the event operations team.

## Post-event reporting

Post-event reports include NPS, session satisfaction, speaker ratings, recurring issues, and suggested improvements for the next edition.
