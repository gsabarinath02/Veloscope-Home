# Support Ticket Policy

## When to create a ticket

Create a support ticket when the assistant cannot answer confidently from approved event documents, when the question is account-specific, or when the user needs help with invoice access, billing, private gallery permissions, login issues, refunds, contracts, or legal requests.

## Ticket response SLA

Critical event-day tickets should be reviewed within 15 minutes. Standard attendee support tickets should be reviewed within four business hours. Sponsor and finance questions should be routed to the event operations owner.

## Ticket information

Every ticket should include the user question, the reason for escalation, the date and time, the source channel, and the last assistant action.
